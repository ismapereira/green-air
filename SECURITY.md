# 🛡️ Política de Segurança (Security Policy)

## Versões Suportadas

Atualmente, apenas a versão estável mais recente do GreenAir recebe correções e atualizações de segurança ativas.

| Versão | Suportada Ativamente |
| ------ | -------------------- |
| v2.2.x | ✅ Sim |
| < v2.2.0 | ❌ Não |

## Como Reportar uma Vulnerabilidade

A integridade do GreenAir e a privacidade dos dados dos nossos utilizadores são fundamentais. Se descobrir uma vulnerabilidade de segurança, **por favor, não abra uma Issue pública**. Divulgar falhas publicamente expõe o projeto a ataques antes que uma correção seja aplicada.

Siga os passos abaixo para efetuar um reporte seguro:

1. Envie um e-mail para **suportegreenair@gmail.com** detalhando o problema.
2. Inclua uma descrição clara da vulnerabilidade, o impacto potencial e os passos exatos ou um exemplo de código (*Proof of Concept*) para reproduzir a falha.
3. Comprometemo-nos a analisar o relatório com a máxima prioridade, responder com uma confirmação e trabalhar numa correção célere.

Obrigado por ajudar a manter a comunidade do Green Air segura de forma responsável!
